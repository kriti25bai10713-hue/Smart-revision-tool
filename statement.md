# Project Statement & Scope

## Problem Statement
Traditional note-taking is passive. Students often struggle to convert their notes into active study materials, leading to poor retention of key concepts. This tool automates that transition.

## Project Scope
This project covers the development of a text-to-flashcard pipeline using basic NLP. It is designed for English language academic notes and focused on noun-phrase extraction.

## Target Users
* University students following flipped classroom models.
* Educators looking to generate quick quizzes from lecture materials.

## High-Level Features
* Sentence-level tokenization.
* POS-tag-based keyword masking.
* Automated study-sheet generation.
